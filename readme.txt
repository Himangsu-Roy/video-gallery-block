== Description ==
## Responsive WordPress Gutenberg Video Gallery Block with albums, filters, captions, and lightbox support.

**Video Gallery Block** is a lightweight Gutenberg plugin for creating professional, responsive video galleries in WordPress.  
Easily add videos to your posts or pages using a modern grid layout. Organize videos with album filters and display them in a popup lightbox — all within the native Gutenberg block editor.  

Showcase **YouTube, Vimeo, Wistia, or self-hosted videos in lightbox** in a sleek gallery layout — no coding required.


**[Demos](https://bblockswp.com/demo/video-gallery)**


= Free Features =

- **Album**: Create and set albums for videos and filter with them.

- **Hide All Album**: Hide the “All” (common) album button by removing its label.

- **Video Thumbnail**: Automatically generate thumbnails from videos.

- **Thumbnail**: Upload a custom thumbnail/poster for each video.

- **Caption**: Add captions for every video in the popup.

- **Layout**: Set columns, gaps, and popup video width with aspect ratio.

- **Gallery Style**: Customize wrapper background, padding, border, and shadow.

- **Filter Buttons Style**: Style filter buttons with typography, colors, and hover effects.

- **Item**: Set the height of gallery items.

- **Titles & Captions**: Add titles, subtitles, and captions to videos.

- **Albums**: Group videos into albums for easy browsing and filtering.

- **Gallery Styles**: Choose from multiple gallery styles for unique layouts.



= Pro Features =

- **Video Sources**: Add videos from YouTube, Vimeo, Wistia, or your own files.

- **Sliders & Carousels**: Showcase videos in sliders, carousels, lightbox galleries, testimonials, and more.

- **Advanced Layout Controls**: Adjust columns, rows, spacing, and gaps for a clean, custom design.

- **Headers & Subheaders**: Add optional headers and subheaders for engaging section titles.

- **Video Player Styling**: Customize video size, play button design, and playback style.

- **Navigation Styling**: Personalize navigation arrows, carousel dots, and thumbnails.

- **Responsive Design**: Apply different layouts for desktop, tablet, and mobile devices.

- **Dimension Controls**: Fine-tune sizes using pixels, percentages, or em units.

- **Interactive Overlays**: Style overlays, badges, and hover effects for each video.

- **Custom Typography**: Style titles, subtitles, ratings, and descriptions with font, size, color, and alignment options.

- **Global Styling**: Control gallery backgrounds, padding, borders, and shadows across all layouts.

- **Advanced Navigation Options**: Enable or disable navigation arrows, dots, captions, or ratings per layout.

- Add the Video Gallery Block from the block category called "Widgets" in the Gutenberg editor.
- You can change block settings from the right-side settings sidebar.

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Video Gallery Block**
3. Click on the **Video Gallery Block** to add the block

= Download & Upload: =
1. Download the **Video Gallery Block** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`Video Gallery-block.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Video Gallery Block** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress



== Frequently Asked Questions ==

= Is Video Gallery Block free? =
Yes! Video Gallery Block is free to use and includes features like albums, captions, thumbnails, star ratings, basic gallery layouts, and styling controls.

= What extra features are included in Pro? =
The Pro version unlocks advanced options like sliders, carousels, lightbox galleries, responsive controls, Wistia groups, custom typography, player styling, and support for multiple video sources (YouTube, Vimeo, Wistia, HTML5).

= Does it work with any WordPress theme? =
Yes, Video Gallery Block works with any modern WordPress theme.

= Can I customize how the gallery looks? =
Absolutely! In the free version, you can adjust columns, gaps, backgrounds, filter buttons, and item height. In Pro, you get full styling control over typography, overlays, navigation, player size, and responsiveness.

= Can I create multiple galleries? =
Yes, you can create and reuse unlimited galleries anywhere on your site.

= Is it mobile-friendly? =
Yes, the free version is responsive by default. The Pro version adds device-specific settings so you can fine-tune layouts separately for desktop, tablet, and mobile.

= Can I use my own video files? =
Yes, you can add videos from YouTube, Vimeo, Wistia, or upload your own HTML5 video files.

= How do I get support? =
Free users can get help in the [support forum](https://wordpress.org/support/plugin/video-gallery-block/). Pro users also receive priority email support and detailed documentation.
